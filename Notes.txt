>>To add product
http://localhost:8080/api/product
 Set method to POST and enter your URL (e.g., http://localhost:8080/product).
Go to the Body tab and select form-data.
First Key:
Key: product
Value: {
  "name": "Sony WH-1000XM5",
  "description": "Wireless Noise Canceling Headphones",
  "brand": "Sony",
  "price": 349.99,
  "category": "Electronics",
  "releaseDate": "2024-05-15",
  "productAvailable": true,
  "stockQuantity": 50
}
Crucial Step: Click the vertical three dots (or hover) at the end of the key row to show the "Content-Type" column. Manually type application/json in that column.
Second Key:
Key: imageFile
Change the Key type dropdown from "Text" to File.
Value: Select an image from your computer.
note :- On the far right of the table headers (where it says Key, Value, Description), click the three vertical dots (or the settings gear).
Check the box that says Content-Type. A new column will appear in your grid.
In that new column, for the product row, type: application/json.
Keep the imageFile row as is (Postman usually handles the file type correctly, but you can set it to image/jpeg or image/png
